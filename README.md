# spin-win-pro

Next.js 14 project scaffold for the "Spin & Win" experience.

How to run:
1. Copy the repository to your machine.
2. Install dependencies: `npm install` or `yarn`.
3. Run dev: `npm run dev` (Next.js will run on default port).
4. Open: `http://localhost:3000/project-2025/fine-dine/spin-win-pro/` (because basePath).

Notes:
- Replace placeholders in `/public/images` and `/public/sounds` with real assets.
- Example .env keys are in `.env.example`.
